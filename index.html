အသုံးပြုလမ်းညွှန်


  1. Customer Name နေရာတွင် Customer ၏နာမည်ထည့်သွင်းပါ။

  2. ဇယားကွက်ထဲတွင်"0"ကိုဖျက်၍ဖြည့်လိုသော Amount များထည့်သွင်းပါ။

  3. အကုန်ဖြည့်၍ပြီးလျှင် Calculate ခလုတ်ကိုနှိပ်၍တွက်ချက်ပါ။

  4. မိမိတွက်ပြီးသော Data များကို Save ခလုတ်ကိုနှိပ်၍ Save ထားပါ။

  5. မိမိ Save ထားသော Data များကို "Saved items" ခလုတ်ကိုနှိပ်၍ပြန်လည်ရယူပါ။

  6. ကျလာသော Box ထဲမှ Customer တစ်ယောက်ချင်းစီ၏ Data များကို၎င်းတို့၏နာမည်ကိုနှိပ်၍ပြန်လည်ရယူ၊ပြင်ဆင်နိုင်သည်။

  7. ပြန်လည်ပြင်ဆင်ပြီးသောDataကိုထပ်မံSaveပါက ယခင် Save ထားသောအရာကိုနာမည်ဘေးမှ "X" ကိုနှိပ်၍ဖျက်နိုင်သည်။

  8. Commission % နှင့် ပေါက်သီးဆကို လိုသလိုပြေင်းလဲနိုင်သည်။

  9. Local storage ကိုသာအသုံးပြုထားသောကြောင့် Browser တစ်ခုထဲကိုသာအသုံးပြုပေးပါ။

  10. ထိုမှသာ မိမိ Save ထားသေား Data များကိုနောက်တစ်ကြိမ်ပြန်လည်အသုံးပြုချိန်တွင် ပြန်လည်ရယူ၍ရမည်ဖြစ်သည်။
